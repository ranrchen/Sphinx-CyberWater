RunModuleAgent

 responsible for the execution of a Model offered as executable.
The model execution requires that the user manually prepares the files required.


[INPUTS]
Ready_List: Connect the output of ForcingDataFileGenerator, AreaWiseParamGenerator and InitialStateFileGenerator if it exists.
WD_Path: WD_Path output port of the MainGenerator. This is the directory where the simulation files are saved.
DataSet_Class: DataSet_Class output port of the MainGenerator.
arg: arguments required for the execution.
exe: executable file, including the execution parameters (not the arguments)
Result_File_Prefix/Name: Either the prefix of all the cells generated by the model. Or the name of the time-series generated.
Result_File_Separator: Separation character in the output files.
Point_output?: (True/False). Check if the output is a single timeseries in contrast of a distributed result in multiple files, once per cell. Default = False.
Header_lines: . Number of lines that will be ignored at the top of the results file(s).
Output_name_01: Name of the output dataset 1.
Output_name_02: Name of the output dataset 2.
Output_name_03: Name of the output dataset 3.
Output_name_04: Name of the output dataset 4.
Output_name_05: Name of the output dataset 5.
File_Pos_01: Column index in the result files (e.g. model results files) for the dataset 1. The counting starts from 0 for the first column, 1 for the second one.
File_Pos_02: Column index in the result files for the dataset 2.
File_Pos_03: Column index in the result files for the dataset 3.
File_Pos_04: Column index in the result files for the dataset 4.
File_Pos_05: Column index in the result files for the dataset 5.
Header_lines: Number of lines at the top of the resulting files that need to be ignored.

[OUTPUTS]
Output01: Output dataset 1.
Output02: Output dataset 2.
Output03: Output dataset 3.
Output04: Output dataset 4.
Output05: Output dataset 5.
